【游戏名称】 连连看

【游戏介绍】 这是一款基于 Cocos 引擎开发的连连看消除小游戏（Cocos Creator 3.8.8）。玩家需要在棋盘中寻找并连接相同图案进行消除，在有限时间或规则条件下尽可能快速完成整盘清理，获得更高分数并进入下一关。

【玩法说明】 游戏中的图案会以棋盘形式排列显示，玩家需要选择两个相同图案进行连接消除，连接路径满足规则时即可成功消除。

道具： 本版本以基础连连看玩法为主，常见辅助功能包含提示、重排等。

1. 选择第一个图案：点击或触控任意一个目标图案作为起点。
2. 选择第二个图案：再选择一个相同图案，系统会判断是否满足连接条件。
3. 成功消除：若连接路径合法（通常可转折且不穿过障碍），两个图案会被消除。
4. 继续匹配：持续清理棋盘中的图案，逐步减少剩余块数。
5. 完成关卡：当棋盘全部清空时，即可完成当前关卡。

操作： 你可以通过鼠标点击（PC 端）或手指触控（移动端）选择图案进行消除操作。游戏为 web-mobile 构建版本，支持浏览器内交互体验。

目标： 在规则允许的时间/步数内完成棋盘清理，尽量用更少操作获得更高效率与更高分数。

Tips： 优先消除边缘或容易形成通路的图案；当局面复杂时可优先寻找可直接连接的组合，减少后续卡局概率。

【技术介绍】 使用基于 Cocos 引擎的 Cocos Creator 3.8.8 开发的 H5 游戏，主要语言为 JavaScript，构建平台为 web-mobile。项目入口为 index.html，运行时通过 SystemJS 加载 cocos-js/cc.js 与资源配置，设计分辨率为 1280 × 720，支持 PC 和移动端浏览器。
