# 连连看（Cocos Creator 3.8.8）运行说明

## 当前已生成内容
- 核心棋盘模型：随机生成、0/1/2 折连线判定、提示搜索、死局重排
- 运行时 UI：棋盘、点击交互、连线显示、计时/分数/提示/洗牌/暂停
- 无素材可直接运行（数字块占位，后续可替换为精灵图）

## 使用方式（空工程）
1. 在 Cocos Creator 3.8.8 打开项目。
2. 打开默认场景（或新建场景）。
3. 确保场景中有 `Canvas` 节点。
4. 将脚本 `assets/scripts/ui/LlkGameRoot.ts` 挂到 `Canvas` 节点。
5. 运行预览（浏览器/H5）。

## 上线前建议（必须做）
1. 将数字块替换为真实美术资源（图标/主题皮肤），做资源分包与压缩。
2. 接入音效（点击、消除、通关、失败）与用户设置（静音/震动）。
3. 接入新手引导、结算页、二次确认退出、复活逻辑（如需要）。
4. 做埋点：开局、通关率、失败原因、使用提示/洗牌次数、停留时长。
5. 做适配与性能测试：低端安卓机、微信/抖音 WebView、横竖屏策略。
6. 增加关卡系统（目标、障碍物、难度曲线）和配置化数据表。
